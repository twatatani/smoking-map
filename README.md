# 喫煙マップ (Smoking Spot Finder)

現在地や指定した場所から、近くの喫煙所を地図上で探せるWebアプリケーションです。

## 公開URL
[https://smoking-map.onrender.com/smoking-points](https://smoking-map.onrender.com/smoking-points)

## 主な機能
- 喫煙所の地図表示
- 新規喫煙所の登録・編集・削除
- 詳細情報の閲覧

## 使用技術
- 言語：Java 21
- フレームワーク：Spring Boot 3
- データベース：PostgreSQL (Render)
- インフラ：Render (Dockerデプロイ)
